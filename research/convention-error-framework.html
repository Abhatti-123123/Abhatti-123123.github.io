Convention Error Framework

This research note connects the detectability paper and the over-differencing paper into a broader framework. The core idea is that returns are a convention: the first difference of the price path. That convention is economically natural, but it also imposes a particular transformation on every asset at every time.

The framework asks what happens when the market's operative memory structure differs from the conventional transform. The working hypothesis is that over-differencing can create artificial autocorrelation, hide part of the memory structure, and reduce the signal-to-noise ratio of the forecasting target.
