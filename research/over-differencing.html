Market Memory Structure and the Cost of Over-Differencing

This paper studies how differencing choices affect machine-learning forecastability. The central empirical question is whether the conventional return transform removes useful memory before a model has a chance to learn it.

The work compares differencing choices across hundreds of S&P 500 constituents using walk-forward forecasting experiments. The core finding is that over-differencing can be more damaging than mild under-differencing because destroyed structure cannot be recovered downstream.
