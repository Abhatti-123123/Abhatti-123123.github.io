Return Innovations vs Returns

A common intuition is that returns are close to innovations: the new information arriving to the market. My research questions whether that equivalence always holds.

If the price path integrates information with persistence, then the return transform may not isolate the primitive innovation. It may produce a transformed object that combines innovation noise with the effects of the differencing operator.

This matters for forecasting. If part of the predictable structure lives in the memory component of the price path, then computing returns before modeling may reduce the signal-to-noise ratio of the target itself.
