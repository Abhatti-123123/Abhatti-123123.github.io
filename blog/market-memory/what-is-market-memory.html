What Does Market Memory Mean?

Market memory is the idea that past information can remain embedded in the price path for longer than a short-memory model would suggest. In a purely short-memory process, autocorrelations decay quickly. In a long-memory process, dependence can decay slowly enough that distant observations still carry information about the present.

In financial markets, this does not automatically mean easy predictability. Memory can come from many sources: slow institutional rebalancing, volatility clustering, liquidity effects, order-flow persistence, or regime changes. The challenge is not just to ask whether memory exists, but to ask what kind of memory is being measured.

My research focuses on the finite-sample problem: what part of that memory is actually detectable in realistic samples, and what happens when we transform it away before modeling begins?
